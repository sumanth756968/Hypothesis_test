# 📊 Hypothesis Testing Using Python

## 📌 Project Overview
This project demonstrates the application of **Hypothesis Testing** using Python to determine whether the average weekly operating cost of Bombay Hospitality Ltd. franchises is higher than the cost predicted by the company's operating cost model. The analysis is performed using a **One-Sample Z-Test** to make statistical decisions based on sample data.

---

## 🎯 Objectives
- Understand the concept of hypothesis testing.
- Define the null and alternative hypotheses.
- Calculate the expected weekly operating cost.
- Perform a One-Sample Z-Test.
- Compare the calculated Z-value with the critical value.
- Interpret the statistical results.
- Draw a business conclusion based on the hypothesis test.

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- NumPy
- SciPy
- Math Library

---

## 📂 Project Workflow
1. Import the required Python libraries.
2. Define the given sample statistics.
3. Calculate the expected operating cost using the cost model.
4. State the null and alternative hypotheses.
5. Compute the Z-test statistic.
6. Determine the critical Z-value at the chosen significance level.
7. Compare the calculated Z-value with the critical value.
8. Interpret the results and make the statistical decision.

## 📈 Conclusion
This project provides a practical understanding of hypothesis testing using Python. It demonstrates how statistical tests can be applied to validate business claims by comparing sample data with expected values. The One-Sample Z-Test helps determine whether sufficient evidence exists to support the restaurant owners' claim regarding increased operating costs.
